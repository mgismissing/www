Why are some users prioritized?

A user is prioritized if it has the @priority role. You can get it by:

  • Buying it
  • Boosting the server
  • Being a moderator or an admin

What advantages do prioritized users have?

Users who have the priority role are the first to be approved for the creation of server resources (forms, ads, etc).
